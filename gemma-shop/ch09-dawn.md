# 尾声 · 第三十二个清晨

> 这章要讲什么?
> 阿衍要回去了. 沈姨在屋顶上让他把整栋楼从头画一遍, 不许翻笔记本. 这一画, 就是从"听过一遍"到"自己能造"的分水岭.

---

## 一

凌晨五点.沈姨敲门.

阿衍已经醒了,坐在书桌前.他没收拾东西——背包从来就没打开过.他只是把笔记本翻到最后一页.

"上屋顶."

---

## 二

四月二十日的横滨清晨.

风比夜里小,但更凉.东边的天还没亮,云有一条很淡的灰白.横滨港的灯一盏一盏在熄,空气里有一股海的咸.

沈姨站在屋顶边上,背对他,和七天前他第一次上来那一晚一模一样.

"画."

阿衍走到她旁边,把笔记本翻到一张空白页,从口袋里掏出那把铜尺——沈姨说过,这把尺以后是他的——把页面四角压住,免得风把纸吹跑.

他闭眼三秒.

然后开始画.

---

阿衍画完,把笔放下.

---

## 四

沈姨没回头看.她只是问:

"重点你抓到了吗."

阿衍想了想.他没看图,他只说了五句话:

> **一**, 译铺优先, 学习副产物.
> **二**, 楼上不该知道楼下的库叫什么.
> **三**, 一份类型走三处, 不重复定义.
> **四**, 量错 token 不是省 token, 是丢最早的那句话.
> **五**, 守夜人不是 cron job, 她是你的纪律.

沈姨这才转过身.

她笑了.

"——画得行.话也对."

她伸手,从风衣口袋里掏出一个小布袋,递给阿衍.阿衍接过来,打开——里面是一张折好的红纸.

红纸上写着一行字:

> **2026-04-25: 阿衍离铺时, 已能从地下室画到屋顶, 不翻笔记.**

阿衍愣了一下.他想起地下室那面卷轴上的蓝胶带——这是同一种笔迹.

"这是给我的?"

"这是你那栋楼的**第一条 KG 三元组**.从今天起, valid_from 是今天, valid_to 是 NULL——只要你那栋楼还在, 这条就一直为真."

---

## 五

七点.阿衍下楼.

他经过二楼,玻璃娃娃屋的灯都熄了,但形状还在那儿.他经过一楼,前台那台旧笔记本上,屏幕亮着一行字——agent 在等他离开,等下一个客人来.

他推门.铜把手在他手心里凉了一下,和七天前一模一样.

外面巷子里有早晨第一拨人,卖早点的、扫地的、收垃圾的.中华街的红灯笼一盏一盏开始熄.

沈姨没下楼送他.她在屋顶,他知道.

---

## 六

新干线.横浜驿,五号车厢,靠窗.

阿衍从背包里取出他的笔记本——不是写字那本,是 13 寸的那台.他没开 IDE,只是新建了一个文件夹:

```
$ mkdir gemma-agent-core
$ cd gemma-agent-core
$ git init
```

他打开 vscode,新建了一个空的 `pyproject.toml`,写了第一行:

```toml
name = "gemma-agent-core"
description = "A translator first, a learner second."
```

然后他打开终端,敲下他这趟旅程的**第一条 commit**:

```
$ git add -A
$ git commit -m "init: scaffold gemma-agent core

Built on a five-floor shop in Yokohama Chinatown.
The watcher knows when not to speak."
```

车窗外,横滨的海消失在晨光里.

阿衍合上电脑,靠回椅背.他想起沈姨最后一句话:

> **"明天你就提交吧.评审看不见这栋楼,他们看见的是码头上停的那艘小船——但你心里这栋楼会一直在."**

他闭上眼.

距离比赛截止,**还有二十四个小时**.

---

## 尾

(扉页背面留白处, 阿衍后来用很小的字补了一行:)

> **写给一年后的自己——**
> 这栋楼不是 hackathon 造的.
> hackathon 只是给了我三十二天, 让我有借口去横滨.
> 真正造楼的, 是沈姨那句"凌晨两点会有人不睡觉".
> 我以后写的每一个 agent, 都要为那个不睡觉的人留一间屋.
